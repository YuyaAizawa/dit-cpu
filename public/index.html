Instruction Set

MomoRISCの実装する命令の一覧です．

形式の欄のRではじまるものはいずれかの汎用レジスタ（A, B, C, D）を表し，Mem[]はメモリアクセスを表します．immは2桁の数値です．

LDI

  • 形式：LDI R imm
  • 疑似コード：R ← imm, PC++

LD

  • 形式：LD R1 R2
  • 疑似コード：R1 ← Mem[R2], PC++

ST

  • 形式：ST R1 R2
  • 疑似コード：Mem[R1] ← R2, PC++

ADD

  • 形式：ADD R1 R2 R3
  • 疑似コード：R1 ← R2 + R3, PC++

R2とR3を足した値をR1に保存します．結果が100以上の場合は100引きます（wrap-around）．

ADI

  • 形式：ADI R1 R2 imm
  • 疑似コード：R1 ← R2 + imm, PC++

R2とimmを足した値をR1に保存します．結果が100以上の場合は100引きます（wrap-around）．

SUB

  • 形式：SUB R1 R2 R3
  • 疑似コード：R1 ← R2 - R3, PC++

R2からR3を引いた値をR1に保存します．結果が0未満の場合は100足します（wrap-around）．

BEQ

  • 形式：BEQ R1 R2 imm
  • 疑似コード：if(R1 = R2) then PC ← imm else PC++

BLT

  • 形式：BLT R1 R2 imm
  • 疑似コード：if(R1 < R2) then PC ← imm else PC++

JPI

  • 形式：JPI R imm
  • 疑似コード：R ← PC + 1, PC ← imm

JP

  • 形式：JP R1 R2
  • 疑似コード：R1 ← PC + 1, PC ← R2

HLT

  • 形式：HLT

実行を止める命令です．最後に自動で追加される他，エラー行にも置かれます．
